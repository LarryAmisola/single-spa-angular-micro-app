Micro Application Angular

Here is a Single-SPA micro application that can also serve as a standalone application.

Background, after converting an angular application to Single-SPA. The application can no longer be serve locally. It display a blank page.

A basic overview to address this concern:

  1. Add a new entry under project in angular.json file.
  2. Add a new file tsconfig.app.json.
  3. Modify package.json file

For a detail information. Please see README file.
